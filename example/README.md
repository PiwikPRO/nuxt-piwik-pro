### Build PiwikPRO Nuxt library

Firstly you need to build `@piwikpro/nuxt-piwik-pro` library:

```bash
npm run build-piwik-nuxt
```

### Setup

Make sure to install the dependencies:

```bash
npm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```
